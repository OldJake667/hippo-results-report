# HiPPO Benchmark Report

Report materials summarizing experiments with HiPPO/state-space memory models and baseline sequence models.

## Overview

This repository contains the written report and selected benchmark outputs from experiments on MNIST, adding, and copying tasks. The work focuses on comparing sequence modeling approaches under a consistent experimental setup.

## My Work

- Ran and summarized PyTorch experiments with HiPPO/state-space memory models.
- Compared recurrence, convolution, attention, and HiPPO/LegS approaches on sequence tasks.
- Wrote the experiment report and prepared the final PDF.

## Files

- `RUN_REPORT.md` - main report in Markdown
- `RUN_REPORT.tex` - LaTeX source
- `RUN_REPORT.pdf` - compiled report
- `benchmark_outputs/` - selected copying-task benchmark results
